Introduction

Materials courses allow you to store course information and resources in a separate Moodle course from the course students are enrolled in. This benefits instructors as they can build their course content and reuse it for all of their sections across semesters.

Return to the top

Terminology

materials course
Moodle course created for an instructor to store course information and content used in teaching. This Moodle course exists separate from other Moodle courses where students and instructors are enrolled and is not associated with any semester.
Return to the top

Restrictions

Only administrators can delete materials courses

Once a Moodle course has been created as a materials course, it can only be deleted by an administrator. It is possible to revoke the request to create a materials course by unselecting the course from the materials courses preferences page before the hourly update occurs.
