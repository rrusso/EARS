Introduction

Course creation and enrollment preferences allow you to determine when your Moodle courses are created and students are enrolled into them. This benefits those who want to have their courses created and students enrolled sooner. There are two types of creation and enrollment preferences: the number of days before the start of classes for Moodle courses to be created and number of days before the start of classes for students to be enrolled into Moodle courses. These preferences are further split into two catagories: default preferences and semester-specific preferences.

Terminology

default preferences
Preferences used whenever there are no semester-specific preferences set for a course.
semester preferences
Preferences set for specific courses within a semester that override the default preferences.
format preferences
Will set the course format upon course creation (see Format).
number of weeks/topics preferences
Will set the number of weeks/topics with the course creation.
availability preferences
Will set the course availability upon course creation (see Availability).
delete preferences
Cleanup preferences. Courses will be deleted when student enrollment equals zero, or at the end of the semester.

Restrictions

Courses must be created before students are enrolled

As there must be a Moodle course for students to be enrolled into, it must be created before students are enrolled. Thus the number of days to create courses must be greater than the number of days to enroll students.

Days must be greater than zero

Creation and Enroll days must be set at a number greater or less than zero.
