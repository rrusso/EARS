How to set default split preferences

In order to set default split preferences, you will be working with the "Default Settings" section of the page as in the picture below:

The default preferences consist of a single form field: whether all courses should be split into individual Moodle courses. By setting this preference to "Yes", all current and future courses will be separated Moodle courses.

Once the page reloads after updating the default preferences, a message should appear stating that your changes were saved. Default preferences will take effect within an hour of being updated. Students and instructors will be unenrolled from any existing Moodle courses and enrolled into their newly created Moodle courses separated from other sections.
