How to cross list courses

In order to cross list courses within one another into a single Moodle course, you must select at least two courses from your pool like the picture below:

Once you have selected the course to cross list together, you must select the number of course shells to create. The default is one course shell.

The next screen then allows to you to move sections freely in the appropriate course shell. A name will be generated for the course shell, but if you have more than one shell, it is highly recommended to give the shells meaningful names.

Once you are satisfied with your changes, submit them, and your new course will be created. If you need to reset your cross list settings, or change the current ones, the process is simple. Select the cross listed courses, and press "Next".

At this screen, you can reset the courses, or return sections back into the pool of sections. You can even add another shell, if the number of sections allow it.

Note: If you have the default delete option set in Course Creation and Enrollment, then the pre-cross listed course will be deleted when enrollment is zero, or more precisely, when all the sections from a course are moved into a course shell or course shells.
