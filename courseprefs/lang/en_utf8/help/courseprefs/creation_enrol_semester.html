How to set semester-specific creation and enrollment preferences

In order to set semester-specific creation and enrollment preferences, you will be working with the forms labeled with the desired semester as in the picture below:

The semester preferences consist multiple courses that you are teaching with fields for the number of days before classes to create Moodle course and to enroll students as illustrated in the picture below. These preferences override the default preferences solely for the courses in the semesters they are set. The same restrictions apply to semester preferences as they do default preferences: courses must be created before students are enrolled.

Once the page reloads after updating the semester preferences, a message should appear stating that your changes were saved. Semester preferences will take effect within an hour of being updated.
