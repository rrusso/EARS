Introduction

Splitting courses allows you to have the sections of a course you teach separated into individual Moodle courses, or course shells. This benefits those who want to customized content for each section and isolate the students from the different sections. This is achieved by applying a split course preference on the desired course. Though splitting your courses has many benefits, it is inevitable that some individuals will want to undo these preferences. Do not worry as it is possible to combine the separate sections of a course into their original Moodle course.

Note: If you have the default delete option set in Course Creation and Enrollment, then the pre-split course will be deleted when enrollment is zero, or more precisely, when a split option is taken.

Terminology

course preference
Preferences set for specific courses stating all sections of the selected course be split into individual Moodle courses.
course shell
A course shell describes a Moodle course created during splitting or cross listing. For example: In a pre-split course with n sections, where n is greater than 1, there is a possibility for 2 to n course shells to be created from the original course.

Restrictions

You must have more than one section in a course.
The split option only makes sense when you have multiple section under the same course number.
No other preference must be applied to the course.
This applies to every preference! Once you have applied any preference (Split, Crosslist, Teamteach), you must revert the course to it's original state before applying a different preference.
