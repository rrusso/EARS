Default creation and enrollment preferences

The default preferences consists of the number of days before classes to create Moodle course and to enroll students. These preferences are used whenever courses do not have semester preferences set. Courses are created 30 days before start of classes and students are enrolled 14 days before start of classes by default.

The rest of the settings can be set manually after a course has been created through Administration->Settings with the exception of Delete option. Each preference will be presented below with it's default.

  • Format: Weekly (see Format)
  • Number of weeks/topics: 17
  • Availability: This course is not available to students (see Availability)

The Delete option cleans up after the teacher. They can choose to delete courses when applying a preference (see Unwanted, Split, Crosslist, Teamteach), or when the semester ends.

Once the page reloads after updating the default preferences, a message should appear stating that your changes were saved. Default preferences will take effect within an hour of being updated.
