ACSA Administration

First, select a sport to assign mentors to. (fig. 1)

Fig. 1

Once a sport is selected, the admin can search for the user to be assigned to that particular sport. A mentor can be assigned to no sports, meaning that the administrator wants this mentor to be assigned to a specific person or people.

Note: see Assigning students to mentors if the admin wants to assign specific people to ASCA mentors of any kind.

Note: the admin can alias the sports code as a human readable name, if he so desires. See Changing the sports name for more details.
